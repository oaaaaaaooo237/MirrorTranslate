# MirrorTranslate
Chrome 翻译插件，如果测试版开启window.AI功能则进行调用，否则转为调用translate google
# MirrorTranslate v1.1 发布说明

**版本**: 1.1.0
**发布日期**: 2025-12-31

## 🚀 重大更新

### 1. 🌐 在线翻译支持 (Online Fallback)
-   **无门槛使用**: 您不再需要安装 Chrome Canary 或开启实验性 AI 标志即可使用。
-   **自动回退**: 当检测不到本地 AI 模型时，插件会自动切换到 **Google Translate API** 进行在线翻译。
-   **真实翻译**: 移除了之前的“模拟模式 (Mock Mode)”，确保所有用户都能获得真实的翻译结果。

### 2. 🎛 全新交互逻辑
-   **工具栏总开关 (Global Master Switch)**:
    -   点击浏览器右上角的扩展图标，可一键 **显示/隐藏** 全局插件。
    -   关闭状态下，页面上的悬浮球会完全消失，且插件停止运行，零资源占用。
-   **页面独立控制**:
    -   点击页面上的“地球仪悬浮球”，仅控制**当前页面的翻译状态**（翻译/还原）。
    -   现在点击悬浮球不会再意外导致图标消失了。

## 🛠 改进与修复
-   **灰色图标状态**: 当插件全局关闭时，工具栏图标会变为灰色并显示 "OFF"，状态一目了然。
-   **状态同步**: 修复了刷新页面后图标状态不同步的问题。
-   **权限更新**: 新增了访问 `translate.googleapis.com` 的权限以支持在线翻译。

## 📦 安装/更新说明
1.  下载并解压 `MirrorTranslate_v1.1.zip`。
2.  在 Chrome 扩展管理页 (`chrome://extensions/`) 点击 **"加载已解压的扩展程序"** (或点击“更新”按钮)。
3.  **注意**: 由于权限变更（新增了网络访问），Chrome 可能会提示重新授权，请点击同意。
